Demo & Showcase for [Waibu MPA](https://github.com/ardhi/waibu-mpa)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install waibu-demo
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```waibu-demo``` in it
